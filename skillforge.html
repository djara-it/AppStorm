01 / 05
BIENVENIDO AL SISTEMA
ARISE es tu sistema de progresión personal. Como en Solo Leveling, el Sistema te guía para forjar habilidades reales. Cada día tienes misiones que cumplir o perderás vida.
02 / 05
MISIONES DEL SISTEMA
Cada día el Sistema genera 3 misiones obligatorias. Pulsa GENERAR para recibirlas. Debes completarlas antes de medianoche o perderás HP. Sin HP el personaje muere.
03 / 05
TEMPORIZADOR POMODORO
Al completar una misión se activa un temporizador de 25 minutos. Si lo completas, la misión se marca como hecha y ganas EXP. Si lo cancelas, no cuenta.
04 / 05
ATRIBUTOS Y RANGOS
Cada tipo de misión sube un atributo distinto. Al subir de nivel el Sistema te asigna un rango: E → D → C → B → A → S. El rango S es el objetivo máximo.
05 / 05
MISIONES PERSONALES
Además de las misiones del Sistema, puedes crear tus propias misiones permanentes. La IA puede sugerirte nuevas según tu perfil y nivel actual.
